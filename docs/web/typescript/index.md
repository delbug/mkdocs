# TypeScript 学习指南

欢迎来到TypeScript学习板块。TypeScript是JavaScript的超集，为JavaScript添加了类型系统。

## 核心概念

- 基本类型
- 接口
- 类
- 泛型
- 装饰器
- 类型推断
- 高级类型

## 最佳实践

- TypeScript配置
- 类型声明文件
- 项目结构
- 与JavaScript互操作
- 错误处理
