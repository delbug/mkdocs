Linux 学习指南

欢迎来到 Linux 学习板块。这里涵盖了 Linux 的核心概念和常用命令。

核心概念

  • 文件系统 - 了解 Linux 的目录结构和文件权限
  • 用户和组 - 管理用户和用户组
  • 进程管理 - 查看和控制进程
  • 网络配置 - 配置 IP 地址、端口和 DNS
  • 包管理器 - 使用 apt、yum 和 dnf 管理软件包

常用命令

文件和目录操作

  • ls - 列出目录内容
  • cd - 切换目录
  • pwd - 显示当前目录
  • mkdir - 创建目录
  • rm - 删除文件或目录
  • cp - 复制文件或目录
  • mv - 移动或重命名文件或目录

文件权限

  • chmod - 修改文件权限
  • chown - 修改文件所有者
  • ls -l - 查看文件权限

进程管理

  • ps - 查看进程
  • top - 实时显示进程信息
  • kill - 终止进程
  • jobs - 查看后台任务
  • fg - 将后台任务切换到前台

网络操作

  • ping - 测试网络连通性
  • curl - 发送 HTTP 请求
  • wget - 下载文件
  • netstat - 查看网络连接
  • ssh - 远程登录

高级特性

  • Shell 脚本 - 编写自动化脚本
  • 系统监控 - 使用 htop、iotop 等工具监控系统
  • 计划任务 - 使用 cron 和 at 定时执行任务
